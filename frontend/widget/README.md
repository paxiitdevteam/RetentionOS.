# RetentionOS Widget

Embeddable widget for intercepting cancel events and showing retention flows.

## Installation

```html
<script src="https://cdn.retentionos.com/widget.js"></script>
<script>
  RetentionOS.init({
    apiKey: 'your-api-key',
    userId: 'user-123',
    plan: 'pro',
    region: 'us'
  });
</script>
```

## Usage

The widget automatically detects cancel button clicks and displays retention flows.
